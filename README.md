# calorizator

Рассчитать КБЖУ по формуле Харриса-Бенедикта.

Для этого в первую очередь надо посчитать уровень метаболизма: 
88.362 + (13.397 * вес в кг) + (4.799 * длина в см) — (5.677 * возраст в годах)

Входные данные:
1. Вес в килограммах хранится в переменной weight
2. Рост в сантиметрах записан в переменную length
3. Возраст в годах записан в переменную age
4. Коэффициент двигательной активности записан в переменную activityRate
